# Cloud Backup and Restore System

A full-stack cloud backup and restore system built with Node.js, MongoDB, and AWS.

## Live Demo
- Frontend: http://cloudvault-frontend-deepsi.s3-website.ap-south-1.amazonaws.com
- Backend: http://3.110.130.238:5000

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Cloud: AWS EC2, AWS S3
- Auth: JWT

## Features
- User Registration & Login
- File Upload to AWS S3
- View & Delete Backups
- Dashboard with Statistics
- Responsive Design
